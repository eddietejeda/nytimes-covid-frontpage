Important Dates
  • December 6, 2019 - The first documented COVID-19 hospital admission.1
  • January 9, 2020 - WHO reported that Chinese authorities determined that the outbreak is caused by a novel coronavirus.2
  • January 28, 2020 - The first time COVID-19 gets top billing on front page.
  • February 11, 2020 - WHO announced that the disease caused would be named COVID-19. 2
What is this site?

I want to track how coverage of COVID-19 changes over time and pin point exactly when the virus stops being front page news. This is a simple implementation that downloads a PDF of the New York Times front page and looks for the following key words: corona, covid, virus, pandemic, wuhan. Check out the source code on Github and let me know if you see any bugs.
